🎓 Финальные Проекты — Июнь 2026

Три полноценных веб-приложения, разработанные студентами в рамках итогового проекта.

Проекты

1. AI Resume Builder
Автор: Айдын Саркыт

Современный конструктор резюме с живым предпросмотром. Пользователь заполняет форму слева — красивое резюме формируется автоматически справа. Поддерживает печать в PDF.

[Перейти в папку →](./resume-builder)

Ключевые возможности:
- Добавление навыков в виде тегов
- Динамические блоки "Опыт работы" и "Образование"
- Живой предпросмотр в реальном времени
- Адаптивный дизайн + оптимизация под печать

---

2. Astana Guide
Автор: Нуржанова Дильяра

Интерактивный гид по Астане с кафе, достопримечательностями и картой.

[Перейти в папку →](./astana-guide)

Ключевые возможности:
- Красивые карточки кафе и достопримечательностей
- Полноценная интерактивная карта на **Leaflet.js** с метками
- Форма предложения нового места
- Адаптивный дизайн (Bootstrap 5)

---

3. Anonymous College Confession Platform (CollegeSecret)
**Автор:** Берик

Анонимная платформа для студенческих признаний, историй и сплетен.

[Перейти в папку →](./college-secret)

Ключевые возможности:
- Полностью анонимная публикация постов
- Фильтры по категориям (Признания / Истории / Слухи)
- Реакции эмодзи в реальном времени
- Красивый современный дизайн

🚀 Как запустить проекты :

Все проекты — **статические** (работают без сервера):

1. Клонируй репозиторий
2. Перейди в папку нужного проекта
3. Открой файл `index.html` в браузере

Или задеплой любой проект на **Vercel / Netlify / Render** (просто перетащи папку).
