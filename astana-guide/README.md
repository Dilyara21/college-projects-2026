# 🌟 Astana Guide — Гид по Астане

**Современный веб-сайт-гид по столице Казахстана**

Проект создан в рамках учебного задания. Полностью соответствует требованиям: содержит информацию о кафе, достопримечательностях, интерактивную карту и систему рейтингов.

# ✨ Основной функционал

- 🍽️ Кафе — карточки популярных заведений с адресами, рейтингами и описаниями
- 🏛️ Достопримечательности — список ключевых мест Астаны с рейтингами
- 🗺️ Интерактивная карта — полноценная карта на базе **Leaflet** с метками всех достопримечательностей
- ⭐ Рейтинги— в изуальное отображение оценок пользователей
- 📱 Адаптивный дизайн** — отлично работает на компьютере и телефоне
- 🎨 Современный UI/UX — красивые карточки, анимации, удобная навигация

# 🚀 Как запустить проект

1. Распакуй архив `astana-guide-pro.zip`
2. Открой папку проекта
3. Дважды кликни по файлу **`index.html`**
4. Сайт откроется в браузере

**Важно:** Для корректной работы карты нужен интернет (первый раз).

# 🛠 Технологии

- HTML5 + CSS3
- Bootstrap 5 — адаптивность и компоненты
- Leaflet.js — интерактивная карта
- Vanilla JavaScript — динамическая загрузка данных
- JSON — хранение информации о местах
